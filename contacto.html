Para Adesão de Serviço

Para aderir a um serviço, basta acessar a página "Serviço", posto lá cadastra-se com os dados do cliente e com os dados do serviço que pretendes adquerir.

Notificações de Novos Serviços

Para a recepção de novos serviços deves seguir os seguintes passos:

1º Acessar a página "Home"

2º Localizar a área "Mantem-se actualizado"

3º Cadastra-se inserindo o seu whatsapp ou email

4º Enviar os dados

Pronto assim está concluido. Quando o site disponibilizar novos serviços, automaticamente irás receber uma notificação via email dizendo que "Novos serviços foram disponibilizado no site".
